# Integrity Risk Assessment Framework

This framework helps reviewers and editors evaluate risk without overclaiming.

## Risk principle
No single signal should drive a misconduct conclusion.

Use independent verification across:
- citation reliability
- methods traceability
- data plausibility
- figure integrity

## Assessment workflow
1. collect observations
2. verify each observation
3. assign confidence level
4. recommend next action proportional to risk

## Action ladder
- Low risk: clarification request
- Medium risk: targeted verification package
- High risk: formal editor escalation with evidence log

## Documentation standard
Each risk note should include:
- observation
- verification method
- objective result
- confidence
