# How to Request Raw Data Professionally

Ask with clear scope:
- Which figure or analysis
- Why it is needed for verification
- Preferred file format


## Verification sequence
1. Verify citation-grounded claims
2. Verify methods traceability
3. Verify data plausibility
4. Report confidence and next action
