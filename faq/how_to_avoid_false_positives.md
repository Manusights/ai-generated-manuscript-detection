# How to Avoid False Positives

- Do not infer misconduct from phrasing
- Verify citations and data claims directly
- Ask for clarifications before conclusions


## Verification sequence
1. Verify citation-grounded claims
2. Verify methods traceability
3. Verify data plausibility
4. Report confidence and next action
