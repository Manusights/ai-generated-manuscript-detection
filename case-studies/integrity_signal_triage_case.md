# Case Study: Integrity Signal Triage

## Scenario
Reviewer notices unusual citation patterns and method vagueness in a high-impact claim section.

## Triage method
1. citation verification sample
2. methods traceability check
3. figure plausibility check
4. confidence labeling and editor summary

## Outcome
- objective evidence log produced
- editor received actionable, non-accusatory report
- decision moved to targeted clarification request
